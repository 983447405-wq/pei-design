# Project Media Slots

Replace the placeholder project content in `index.html` with real files using these paths:

- `assets/projects/images/` for PNG/JPG/WebP screenshots and diagrams
- `assets/projects/pdf/` for case study PDFs and resume PDFs
- `assets/projects/videos/` for MP4/WebM prototype walkthroughs

The current homepage already contains project cards and modal entrances for image, PDF, and video project types.
